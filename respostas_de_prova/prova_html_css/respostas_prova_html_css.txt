Como é formada uma regra CSS?
R:Por um seletor ou um grupo de seletores e por pelo menos uma declaração, que é formada por uma prioridade e um valor

Qual dessas declarações da propriedade padding está correta?
R: padding: 11px 11px;

Como podemos alinhar um texto na tela?
R: Com a propriedade text-align, usando os valores left, right, center e justify

Para que serve o border-radius?
R: Para arredondar cantos de elementos

Quais são as 4 áreas do box-model?
R: Content, padding, border e margin

Qual é o valor que colocamos na propriedade margin para nos ajudar a alinhar elementos horizontalmente?
R: Auto

É possível adicionar imagens aos marcadores de uma lista?
R: Sim, com a propriedade list-style-image
